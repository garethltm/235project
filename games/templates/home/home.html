This is a game library website created for the 2023 S2 version of Computer Science 235 paper at the University of Auckland in New Zealand.
This website is work in progress by 3 students, Minsu Kim, Aaron Chiam, and Gareth Lee.
Currently, this website allows users to browse, view game details and search by the game name, genre and publisher.
At the moment, this website reflects the requirements of CS235 Assignment Phase 2.
Game data has been provided by our lecturer from Steam, in the form of a CSV file.
